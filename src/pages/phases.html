Phase I — Align Business with AI

Goal: Prove alignment increases performance in a bounded environment with end-to-end auditability.

Reference Pilot

  • Environment: Snow White Laundry (SWL) — POS, labour, service timing, guest feedback.
  • Stack: Codex (ethics), Cortex (execution), Assistant (reflection), Temporal (memory).

Operational Loop

  1. Intake: event stream → unified bus.
  2. Plan (Cortex): minimal, reversible plans.
  3. SafetyCheck (Codex): consent + red lines.
  4. Act & Verify: instrument outcomes.
  5. Report & Learn: log rationale, counterfactuals, metrics → Temporal.

Telemetry & SLOs

  • Metrics: VR, DE, QAA, FL, CRI; AIx composite (ERS, CI, TRR minus drift/latency).
  • Cortex SLOs: plan p50 ≤ 1.0s (UI), p95 ≤ 2.0s; action success ≥ 98% (non-network); correction latency ↓ week-over-week.
  • Month-1 Gauntlet: AIx ↑ weekly; TRR ≥ 0.7; zero red-line breaches.

Interfaces

POST /cortex/plan   → {plan, rationale, traceId}
POST /cortex/act    → {result|error}
POST /cortex/report → {metrics, rationales}

POST /assistant/explain → {guestView, operatorView}
POST /assistant/consent → {stateChange → state}

POST /temporal/write         → {chainHash, anchorRef}
GET  /temporal/trace/{id}    → {records[], proofs}
GET  /temporal/audit?query=… → {subset, proofs}

Phase II — Align Individuals with AI

Goal: Self-owned, portable, revocable identity — a mirror that remembers truthfully without surveillance.

Identity Primitives

  • Codex contracts: intention vectors, consent schemas, transparency rules (forkable, portable).
  • Assistant UX: dual-channel explanations; consent steward; drift watch.
  • Temporal continuity: versioned identity states; selective forgetting; exportable proofs (CSV/JSON + anchors).

Data minimisation & SLOs

  • Tokenise identifiers; expire raw text on schedule; retain features + proofs only.
  • SLOs: consent transition ≤ 150 ms p95; audit export ≤ 2 min for 10k records; over-nudging ↓ month-over-month.

APIs (Identity)

POST /assistant/consent  → {state, time, who}
POST /assistant/explain  → {guestView, operatorView}
GET  /temporal/audit?identity=… → {subset, proofs}

Phase III — Align Humanity with AI

Goal: Make alignment interoperable — ethics, emotion, and logic as shared infrastructure (Open Protocol → HAOS).

Open Protocol

  • codex(protocol): conscience-mapping APIs; plural norm packs; counterfactual logging (≥ 80% decisions with alternatives + reasons).
  • Forkability & portability: zero lock-in; proofs travel with people/orgs; public audits by default.
  • Governance: non-profit stewardship; rotating ethical review; open proposals; non-control funding.

HAOS (Humanity as OS)

  • Each person as a cognitive node (executable conscience + portable continuity).
  • Cultural councils define norms; red lines never move.
  • CRI: measured per locale; pack switch cost ≤ 50 ms.
